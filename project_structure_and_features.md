# GuessFunc (猜函数游戏) 详细功能说明文档

《GuessFunc》是一款将数学函数与解谜推理相结合的视觉游戏。玩家无需具备深厚的数学推导能力，只需通过观察给定的函数图像，利用直觉和数学知识，在输入框中构造出相应的函数表达式，使其与目标曲线完美重合即可过关。

以下是该游戏支持的全部功能模块及其详细描述：

## 一、 核心游玩功能 (Game Play)

### 1. 函数图像可视化
- **实时渲染交互**：游戏主界面提供了一个完整的可交互坐标系（支持拖拽平移、滚轮缩放），目标函数曲线将以黑色粗线展示在图表上，玩家输入的公式将以红色线条实时呈现。
- **动态拟合反馈**：当玩家修改输入的数学表达式时，坐标系中的红色曲线会瞬间更新，玩家可以直观地看到自己的输入与目标曲线的偏差，从而不断调整参数和表达式。
- **参数滑块控制**：在某些复杂的关卡中，系统会提供预设的变量（如 $a, b, c$ 等），并在界面上生成对应的滑动条。玩家可以通过拖动滑块，实时观察参数变化对函数图像形态的影响，辅助破局。

### 2. 智能等价验证引擎
- **非死板匹配**：玩家过关不需要输入与标准答案“字符串完全一致”的公式。只要数学上等价即可通关（例如，目标是 $2x$，玩家输入 $x+x$ 或 $\frac{4x}{2}$ 都会被判定为正确）。
- **多维度支持**：支持普通函数、隐式方程（如 $x^2+y^2=1$）、不等式以及包含各类数学符号（三角函数、对数、指数、绝对值等）的验证。

## 二、 游戏模式 (Game Modes)

### 1. 故事模式 (Story Mode)
- **剧情推进体验**：游戏不仅是单纯的数学题，还包含了一条主线剧情。玩家在解开特定关卡后，会解锁对应的剧情片段（如聊天记录、邮件、文档备忘录或音频文件）。
- **进度锁与章节系统**：关卡按“路线 -> 章节 -> 关卡”的树状结构组织。玩家必须完成前置关卡才能解锁后续内容，防止跳关。
- **沉浸式阅读器**：内置了多种剧情文件阅读器，支持类似视觉小说 (Visual Novel) 的沉浸式文本阅读和剧情演绎。

### 2. 随机挑战模式 (Random Challenge)
- **无尽的题库**：系统可以根据玩家设定的难度，自动生成无穷无尽的随机数学函数。
- **难度平滑调节**：玩家可以自定义生成的难度级别，决定是否在随机函数中包含复杂的参数。系统会根据难度智能组合多项式、三角函数等元素，生成具有挑战性的题目。

### 3. 自由创作与分享 (Custom & Share)
- **链接分享**：玩家可以随时在游玩界面生成一个当前关卡的“挑战链接”（将目标函数编码到 URL 中），直接发送给朋友，对方点击链接即可立刻在浏览器中游玩该关卡，无需安装任何模组。

## 三、 创意工坊与模组系统 (Workshop & Modding)

### 1. 模组管理器 (Mod Store)
- **本地模组库**：玩家可以下载、安装、启用或禁用不同的玩家自制关卡包（模组）。
- **无缝切换**：在模组管理器中激活的模组，会自动替换或追加到主界面的关卡选择列表中，玩家可以像游玩官方故事一样游玩玩家自制内容。
- **导入与导出**：支持将自己制作的模组一键打包导出为存档文件（如 JSON），或从外部导入他人的模组文件。

### 2. 关卡编辑器 (Story Editor)
- **全屏工作区**：提供了一个类似于专业 IDE 的全屏编辑环境，左侧为章节树状目录，右侧为编辑区。
- **可视化编辑**：
  - **路线与章节管理**：可以随意新建、重命名、拖拽排序章节和路线。
  - **关卡设计**：创作者可以直接输入目标函数的表达式，设定难度，甚至通过内置的“调试面板”实时预览该函数在游戏内的表现。
  - **剧情文件编写**：支持编写 Markdown 格式的剧情文本，支持配置对话记录、系统日志等丰富的故事表现形式。
- **批量生成器**：为了减轻创作者的负担，编辑器内置了批量关卡生成工具，可以一次性生成指定数量和难度的随机关卡填充到章节中。
- **实时保存与防丢**：创作者在编辑器内的所有修改都会在几秒内自动保存到本地存储中，即使意外关闭网页也不会丢失进度，并且支持撤销/重做操作。

## 四、 辅助与偏好设置功能 (Settings & Accessibility)

### 1. 个性化界面与主题
- **多语言支持**：原生支持简体中文和英文的无缝切换，界面文本、剧情内容均会同步改变。
- **亮/暗色模式**：支持全局跟随系统或手动指定的明亮模式与暗黑模式，保护视力。
- **字体切换**：允许玩家根据阅读偏好，在不同的界面字体间切换。

### 2. 数据与存档管理
- **自动云端/本地存档**：所有的通关进度、已读剧情、设置偏好都会自动保存在浏览器本地，下次打开无缝衔接。
- **存档导出/导入**：玩家可以在设置中将完整的游戏进度导出为一个文件进行备份，或在另一台设备上导入该文件恢复进度。
- **进度重置**：如果想重新体验剧情，支持一键清空所有游戏进度（保留设置）。

### 3. 辅助功能 (Accessibility)
- **护航模式 (Assist Mode)**：如果玩家卡关，可以开启辅助模式，解除所有的剧情锁和关卡锁，允许玩家自由游玩任意一关。
- **侧边栏折叠**：在游玩界面，玩家可以随时折叠或展开左侧的菜单栏，以获得更大、更纯粹的函数图像观察视野。

## 五、 音频与反馈系统
- **背景音乐控制**：内置符合游戏氛围的背景音乐（如钢琴曲、史诗音乐等），玩家可以在全局顶部栏随时控制播放/暂停，并在设置中调节音量。
- **视觉反馈**：当玩家成功解答出一个极具挑战性的函数时，系统会触发全屏的五彩纸屑 (Confetti) 庆祝动画，提供正向的情绪反馈。
- **错误提示**：当输入的表达式语法错误或缺乏方程式时，系统会弹出友好的提示框指导玩家修正输入。
