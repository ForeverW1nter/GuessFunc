# 猜函数 (Guess the Function) 游玩规则

欢迎来到猜函数挑战！这是一个锻炼数学直觉与代数推导能力的趣味小游戏。

## 核心规则

1. **观察目标图象**
   - 页面绘图区域中显示的 **黑色粗实线** 即为目标函数 $f(x)$。你的任务是推导出它的解析式。

2. **侧边栏输入**
   - 点击左侧的 Desmos 列表框（或展开侧边栏）来输入你猜测的函数。
   - 你可以输入多行函数，系统会实时渲染你输入的图象以便对比。

3. **利用引用技巧**
   - 你可以在输入中使用 `f(x)` 来辅助推导。
   - **求导：** 输入 `f'(x)` 可以观察目标函数的导数图象。
   - **平移：** 输入 `f(x-1)` 或 `f(x)+2` 来观察平移变换。
   - **缩放：** 输入 `f(2x)` 或 `2f(x)` 来观察伸缩变换。

4. **判定获胜**
   - 当你觉得某一行表达式与目标函数完全重合时，点击右上角的 **“确认猜测”** 按钮。
   - **限制条件：** 参与判定的表达式中 **不能包含字母 `f`**（即不能直接引用目标函数本身获胜）。
   - **语法支持：** 支持 `2x` (省略乘号)、`sin x`、`|x|`、`\sqrt{x}`、`\frac{a}{b}` 等标准数学语法。

## 获胜反馈
- **猜对了：** 顶部弹出绿色气泡 “太棒了！你猜对了！🎉”，1.5秒后自动消失。
- **猜错了：** 顶部弹出红色气泡 “猜错咯，再试试吧！🤔”，1.5秒后自动消失。

---
*提示：如果遇到极其复杂的函数，可以尝试通过特殊点（如 $x=0, 1$）的值以及对称性来寻找突破口。*
