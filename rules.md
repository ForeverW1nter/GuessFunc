# 游戏规则

## 1. 游戏目标
你的目标是猜出面板上黑色曲线所代表的函数表达式 $f(x)$。
当你的表达式与目标函数等价时，即可获得胜利！

## 2. 如何操作
1.  **观察曲线**：仔细观察黑色曲线的形状、零点、渐近线等特征。
2.  **输入猜测**：在左侧面板的**第一行**中输入你猜测的函数表达式（例如 $y = x^2$ 或 $\sin(x)$），其他行输入想观察与对比的函数表达式。
3.  **辅助验证**：你可以输入 $f(2)$ 等具体数值来计算目标函数在特定点的值，也可以输入类似 $f'(x+1)+2$ 的式子来观察目标函数变换后的图象。
4.  **确认猜测**：点击顶部的 **“确认猜测”** 按钮提交你的答案。

## 3. 判定标准
系统会自动判定你输入的表达式与目标函数是否等价。
-   支持四则运算
-   支持初等函数 $\sin x, \cos x, \tan x, \exp x, \ln x, \sqrt{x}, |x|$ 等
-   只要数学意义上等价即可（例如 $x+x$ 等价于 $2x$）
-   不要使用分段函数、递归函数或极坐标，尽管 Desmos 支持这些功能。

## 4. 其他功能
-   **随机关卡**：随机生成一个新的挑战。
-   **创建关卡**：在**第一行**输入一个函数，点击创建，生成你自己的关卡。
-   **分享关卡**：复制当前关卡的链接分享给好友挑战。
