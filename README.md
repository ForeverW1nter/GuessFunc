## 什么是 GuessFunc？

GuessFunc 是一款基于数学函数的可视化解谜游戏。在这里，你不必是数学天才，只需通过观察和直觉，去匹配那些隐藏在数据背后的数学规律。

## 游戏模式

1.  **闯关模式 (Story Mode)**
    *   随着故事的深入，真相将被一层层剥开……结局由你来见证。
    *   当前版本剧情为 AI 生成，旧版线路“The Day Before Tomorrow”关卡为 AI 生成，新版线路“See You Tomorrow”并非最终版本。最终版本的剧情与关卡将在后续更新中发布。（Coming S∞n）

2.  **随机挑战 (Random Challenge)**
    *   无限生成的随机函数，提供平滑难度选择。
    *   挑战你的极限，看看你能解出多复杂的函数。
